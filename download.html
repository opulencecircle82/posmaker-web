How to Install

Step 1

Download the APK

Tap the download button above for your role. The APK file will save to your Downloads folder.

Step 2

Allow unknown apps

Go to Settings → Security → Install unknown apps, then allow installs from your browser or file manager.

Step 3

Install & open

Tap the downloaded APK file and follow the on-screen prompts. Once installed, open the app and sign in with your mobile number.
