Cashier → Manager → Owner
~60 seconds · English · Screen recording walkthrough
0:00 – 0:20
🧾
Role 1

The Cashier

1
Log in with PIN
"The cashier starts their shift by logging in with their personal PIN — no shared passwords."
Screen: PIN pad, cashier name appears → dashboard loads
2
Add items & process orders
"They select items, choose the payment method — Cash or GCash — then confirm the order."
Screen: Add items → select payment → Checkout → order confirmed
3
End of shift → Remit to Manager
"When the shift ends, the cashier enters their Cash on Hand — the actual cash in the drawer — then submits it to the manager."
Screen: Logout → Enter Cash on Hand → Confirm Remittance → logged out
↓

Cashier's remittance is now waiting for Manager review

0:20 – 0:42
📋
Role 2

The Manager

1
Receive & confirm remittance
"The manager opens the remittance, counts the cash, and confirms it. The system automatically shows if it's Balanced, Short, or Over."
Screen: Remittance card → Cash Sales · Online Sales · COH → Balanced / Short / Over
2
Order stocks via Manager POS
"Only the manager can reorder stocks — this keeps inventory controlled. They select items that are running low and submit the order."
Screen: Manager POS → Stocks tab → select items → Order confirmed
3
Complete assigned tasks
"The owner assigns tasks to the manager — like restocking, cleaning, or reports. The manager marks them done when finished."
Screen: Tasks list → checkbox → marked Complete
↓

Everything is recorded. Now the Owner validates from anywhere.

0:42 – 1:00
📊
Role 3

The Owner

1
Check the Dashboard — anywhere, anytime
"The owner opens the dashboard on any device. No need to be at the store — just an internet connection."
Screen: Phone or laptop → POSMaker dashboard → overview cards (Sales today, Remittances, Active cashiers)
2
Review Activity Log & Remittances
"Every transaction, every login, every remittance — it's all logged. The owner can see who did what and when."
Screen: Activity Log → remittance entries with Short/Over/Balanced badges · cashier name · amount
3
Assign new tasks & monitor performance
"The owner can add tasks for the manager right from the dashboard — and track whether they're getting done. Full visibility, zero guessing."
Screen: Add task → assign to manager → task appears in manager's list
